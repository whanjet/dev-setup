# dev-setup

## Overview
A simple example for setting up a development environment. This repository provides basic configurations to help you get started quickly.

## Features
- Pre-configured settings for development
- Easy to customize

## Technology Stack
- Bash
- Git

## Usage
Clone the repository and follow the instructions in the setup script:

```bash
git clone https://github.com/whanjet/dev-setup.git
cd dev-setup
./setup.sh
```
